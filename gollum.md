# gollum

## 導入背景/目的
- 学習記録の一覧性を高めたい
- 学習記録の編集を容易にしたい
- 学習記録の管理・編集コストを下げたい

<br>
i.e. ノートPCに個人wikiを導入したい

## 何ができる
- ローカル環境で個人wikiを管理できる
- markdown記法が使えるので、編集コストを下げつつ構造的な文章が作成できる
- mathjaxが使えるので、TeXライクに数式を記述できる

## 導入環境

| カテゴリ | 導入環境 |
----|---- 
| PC | MacBook |
| OS | macOS Catalina |
| プロセッサ | 1.2GHz |
